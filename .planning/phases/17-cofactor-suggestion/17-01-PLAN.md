`estimateFeature(String feature, double[] rawValues)`:
1. Drop NaN into a `double[] finite` (percentileSorted does NOT filter NaN ‚Äî see Pitfall 3).
2. `Arrays.sort(finite)` (ascending ‚Äî percentileSorted requires it).
3. `int n = finite.length;` If `n == 0` return `new FeatureCofactor(feature, 0, Double.NaN, Double.NaN, Double.NaN, NEUTRAL_FALLBACK, true, "no finite values")` ‚Äî NEVER throw.
4. `double background = ImagePixelStats.percentileSorted(finite, BACKGROUND_PERCENTILE);` (D-01/D-02: the SAME 50.0 for every feature).
5. `double median = background;` (same p50 statistic ‚Äî the table's median column reuses it per D-09/Research A2).
6. `double p99 = ImagePixelStats.percentileSorted(finite, 99.0);`
7. `double spread = finite[n-1] - finite[0];`
8. Dead/saturated (D-11, mirroring PixelCohortAnalyzer's exclusion intent):
   - `boolean saturated = background >= MAX_COFACTOR;` reason "saturated (background at range ceiling)".
   - `boolean dead = !saturated && spread < DEAD_EPS;` reason "dead (near-zero variance)".
   - `boolean excluded = dead || saturated;`
9. `double cofactor = clamp(background);` where `clamp(v) = Math.max(MIN_COFACTOR, Math.min(MAX_COFACTOR, v));`
10. Return the populated `FeatureCofactor` with the chosen `reason` ("" when not excluded).

`estimate(String[] featureNames, double[][] columns)`:
- For each i, `FeatureCofactor fc = estimateFeature(featureNames[i], columns[i]);` collect into a `List`.
- `double global = globalCofactor(list);` return `new CofactorSuggestion(list, global)`.

`globalCofactor(List<FeatureCofactor> results)` (COF-06/D-10):
- Collect `cofactor()` of rows where `!excluded()` into `double[] keep`.
- If `keep.length == 0` return `NEUTRAL_FALLBACK` (1.0).
- Return `clamp(RobustStats.median(keep));`.
