**apply() method** (single image):
```java
public int apply(qupath.lib.images.ImageData<?> imageData,
                 List<String> markerNames,
                 qupath.lib.projects.Project<?> project,
                 Consumer<String> log) throws Exception
```
- `markerNames` are the user-selected marker names (unsanitized display names from registry).
- `log` may be null — default to no-op.
- Steps:
  1. Get all detection objects: `imageData.getHierarchy().getDetectionObjects()`. If empty, log warning, return 0.
  2. Sort `markerNames` alphabetically (copy first — do not mutate caller's list): `List<String> sorted = new ArrayList<>(markerNames); Collections.sort(sorted);`
  3. Build a `CellFeatureExtractor` from the first detection: `new CellFeatureExtractor(detections.iterator().next())`.
  4. Extract feature matrix once: `float[] flatData = extractor.extractMatrix(detections)`. Store `nFeatures = extractor.nFeatures()`, `nCells` = detection count.
  5. For each marker in `sorted`: sanitize name with `BinaryClassifierRegistry.sanitizeMarkerName(marker)`, load state with `ProjectStateManager.loadBinaryState(project, sanitizedName)`. If null or missing model bytes: skip (log debug), continue. Otherwise load `XGBoostModel.fromBytes(xgbBytes)` and `LightGBMModel.fromBytes(lgbBytes)`. Predict both: `float[][] xgbProbs = xgbModel.predict(flatData, nCells, nFeatures)` and `float[][] lgbProbs = lgbModel.predict(flatData, nCells, nFeatures)`. Store per-cell avg probability for class index 1 (positive class). If class index 1 doesn't exist (classNames.size() < 2): skip marker.
  6. Build `List<PathObject>` and `List<PathClass>` for batch apply. For each detection `i`: iterate over sorted markers that had valid predictions, determine `+/-` suffix using `avgProb[i] >= 0.5f`. Build composite name by joining sorted marker tokens with `:`. If no markers contributed: use `PathClass.NULL`. Else `PathClass.fromString(compositeName)`.
  7. Batch apply: `Platform.runLater(() -> { for each pair: obj.setPathClass(cls); }); imageData.getHierarchy().fireObjectClassificationsChangedEvent(this, detections);` — fire the event ALSO inside `Platform.runLater()` after the setPathClass loop.
  8. Return `nCells` classified.
- Update `progress` and `status` properties (via `Platform.runLater(() -> { progress.set(pct); status.set(msg); })`) at key steps: start (0.0), after feature extraction (0.2), per-marker as fraction (0.2–0.8), after batch apply (1.0).

**batch() method** (multi-image):
```java
public Map<String, String> batch(qupath.lib.projects.Project<?> project,
                                 List<String> imageNames,
                                 List<String> markerNames,
                                 Consumer<String> log) throws Exception
```
- Returns `LinkedHashMap<String, String>`: imageName → result string ("Classified N cells" / "Skipped: no detections" / "Error: <msg>").
- For each `imageName` in `imageNames`:
  1. Find entry: `project.getImageList()` stream filter by `entry.getImageName().equals(imageName)`. If not found: put "Skipped: image not found in project", continue.
  2. Open imageData: `entry.readImageData()`. Null-check.
  3. Call `apply(imageData, markerNames, project, log)`. Store result.
  4. Save imageData back: `entry.saveImageData(imageData)` (persists PathClass changes).
  5. On exception: catch, log, put "Error: " + ex.getMessage(), continue.
  6. Update progress per image.
