  <action>
Create `src/main/java/qupath/ext/celltune/ui/CompositeClassificationDialog.java`.

Package: `qupath.ext.celltune.ui`

**Class:** `public class CompositeClassificationDialog` — NOT a VBox; it manages a `Stage`.

**Constructor:** `public CompositeClassificationDialog(QuPathGUI qupath)`
- Store `qupath` reference as a field.

**showAndWait() method:** `public void showAndWait()` — builds and shows the dialog, blocks until closed.
- Build a `Stage` with `initModality(Modality.APPLICATION_MODAL)`, `initOwner(qupath.getStage())`, title `"Composite Classification"`.

**Dialog layout:**

```
Title label: "Composite Classification"
Subtitle: "Select trained binary classifiers to combine into a composite PathClass."

[CheckBox list — one row per registry marker]
  Each row: CheckBox with text = "MarkerName  ✓ Trained" (or "MarkerName  (untrained)")
  Untrained → checkbox.setDisable(true)
  Styling: trained indicator green text, untrained grey

[Explanatory label — shown only when no trained classifiers exist]
  Text: "No trained classifiers available. Train binary classifiers first."
  Style: -fx-text-fill: #CC0000;

[Button bar]
  [Apply]    [Batch...]

[Result label — initially empty, updated after apply]
```

**Population logic:**
1. Load registry: `BinaryClassifierRegistry.load(project)` → `Map<String, String>`. If null/exception: show empty list.
2. Load saved selection: `ProjectStateManager.loadCompositeConfig(project)` → `List<String> savedSelected`.
3. For each registry entry: check if trained (`ProjectStateManager.loadBinaryState(project, key)` != null and xgboostModelBase64 != null). Create `CheckBox(markerName)`. If trained: set selected = (savedSelected.contains(key)); else setDisable(true), setSelected(false).
4. If no trained classifiers found: show explanatory label (visible=true, managed=true); disable Apply and Batch... buttons.
5. Build a `VBox` with the checkboxes inside a `ScrollPane` (max height 250).

**handleApply() — runs on FX thread initially, delegates work to background Task:**
1. Collect selected marker names from checkboxes.
2. Save config: `ProjectStateManager.saveCompositeConfig(project, selectedMarkers)`.
3. Open progress dialog (Modality.NONE).
4. Create `CompositeClassifier cc = new CompositeClassifier()`. Bind progress dialog bar and status label.
5. Create `javafx.concurrent.Task<Integer>` that calls `cc.apply(imageData, selectedMarkers, project, log::append)`.
6. `task.setOnSucceeded(e -> Platform.runLater(() -> { progressStage.close(); resultLabel.setText("Classified " + task.getValue() + " cells."); }))`.
7. `task.setOnFailed(e -> Platform.runLater(() -> { progressStage.close(); Dialogs.showErrorMessage("Composite Classification", task.getException().getMessage()); }))`.
8. `new Thread(task, "composite-classify").start()`.

**handleBatch():**
1. Collect selected marker names.
2. Save config.
3. Open `ImageSelectionPane`: get all image names from `qupath.getProject().getImageList()`, current image name from `qupath.getImageData()`. Call `showAndWait()`. If result null: return.
4. Open progress dialog (Modality.NONE).
5. Create `CompositeClassifier cc`. Bind progress dialog.
6. Create `Task<Map<String,String>>` that calls `cc.batch(project, imageNames, selectedMarkers, log::append)`.
7. On succeeded: close progress stage; show batch results in a scrollable TextArea dialog or update log.
8. On failed: close progress stage; show error dialog.
9. Start on new Thread.

**Null guards:**
- If `qupath.getProject()` is null: `Dialogs.showErrorMessage("Composite Classification", "No project is open."); return;` — check at the start of `showAndWait()`.
- If `qupath.getImageData()` is null (for Apply): show error, return.

**Required imports:** `javafx.application.Platform`, `javafx.concurrent.Task`, `javafx.geometry.Insets`, `javafx.scene.Scene`, `javafx.scene.control.*`, `javafx.scene.layout.*`, `javafx.stage.*`, `qupath.ext.celltune.classifier.CompositeClassifier`, `qupath.ext.celltune.io.BinaryClassifierRegistry`, `qupath.ext.celltune.io.ProjectStateManager`, `qupath.fx.dialogs.Dialogs`, `qupath.lib.gui.QuPathGUI`, `java.util.*`.
